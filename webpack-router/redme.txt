Задание:
1. Настроить Webpack для использования React;

2. Декомпозировать страницы на компоненты.

Создаем страницу "Частые вопросы".

В файле webpack.config.js задаем точку входа './src/qustions.js', а в объекте output задаем путь к созданию и наименованию папки
и название файла на выходе 'qustionsBundle.js'. В консоле Git CMD вызываем команду "npm start" в соответствии со скриптом
 из файла package.json. Получаем папку dist и в ней соответствующий файл с расширением js и файл index.html. Далее набираем 
команду "npm run server". И получаем на сервере с портом 3006 страницу html "Частые вопросы". 
Ранее уже была набрана команда "npm start", поэтому вводим только команду "npm run server".


Создаем страницу "О нас".

Порядок действий такой же как и написанный выше, только в файле  webpack.config.js меняем точку входа на './src/index.js' и 
меняем файл на выходе на 'bundle.js'. Так же набираем "npm start" и "npm run server". Для облегчения понимания соответствующие
файлы закоментированны.
